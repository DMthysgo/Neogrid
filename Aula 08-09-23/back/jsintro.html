Introdução ao JavaScript

Nesta lição, vamos aprender o básico do JavaScript.

1º topico: O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. Sendo usada principalmente para criar interatividade em páginas da web.

Observação: Diferentemente de Java, que é uma linguagem de programação independente de plataforma, JavaScript é executado no navegador do cliente.

Possiveis questionamentos desse tópico:

O que é uma linguagem de programação de alto nível?

Uma linguagem de programação de alto nível é uma linguagem de programação que é projetada para ser mais fácil de entender e usar para os seres humanos. Se afastando da linguagem da máquina, tendo como caracteristicas chaves, a abstração, a portabilidade, a facilidade de manutenção, e muitas vezes com diversas bibliotecas e frameworks.

O que é tipagem dinâmica fraca?

Aqui temos dois conceitos diferentes de tipagem, que se referem à forma como as linguagens de programação tratam os tipos de dados.

Na tipagem Dinâmica os tipos de dados são verificados em tempo de execução, o que significa que o interpretador ou compilador verifica os tipos de dados durante a execução do programa, à medida que as operações são realizadas.

Na tipagem Fraca as conversões implícitas entre tipos de dados são permitidas e podem ocorrer sem gerar erros. Isso significa que as operações entre tipos de dados diferentes são tratadas pelo sistema de forma a tentar realizar a operação, mesmo que isso possa não ser semanticamente válido.

Por exemplo, graças a tipagem dinâmica é possivel que no JavaScript, possamos atribuir uma string a uma variável e, em seguida, atribuir um número a essa mesma variável posteriormente no código, e o JavaScript não gera um erro até que a operação seja executada.

E por exemplo, graças a tipagem fraca é possivel somar uma string a um número, e o JavaScript tentará converter automaticamente a string em um número para realizar a operação.

Opostas as essas duas tipagens temos linguagens com "tipagem estática" e "tipagem forte" que exigem a declaração explicita dos tipos de dados e não permitem conversões automáticas entre tipos, tornando o código mais rigoroso em relação aos tipos de dados.

2º topico: A Tag <script>

Para adicionar JavaScript a uma página web, usamos a tag <script>. Ela pode ser colocada no <head> ou no <body> do documento HTML. Aqui está um exemplo:

<script>
// Seu código JavaScript aqui
</script>

Neste exemplo temos um script interno ao HTML, com a abertura da tag script, um comentario em linha e o fechamento da tag.

Assim como o CSS que colocamos na tag <style> podendo ser interno ou externo, aqui acontece o mesmo com o JavaScript, um detalhe extra é que podemos ter diversas tags <script> no nosso código.

Para criar um arquivo JavaScript externo é importante saber que arquivos JS possuem a extensão .js, por exemplo temos um arquivo chamado "meuScript.js" e decidimos chama-lo em nosso código HTML, para isso fazemos o seguinte:

<script src="meuScript.js"> </script>

Neste exemplo abrimos a tag script com o atributo source(do inglês: fonte) que se resume na sigla src, esse atributo inclui o arquivo "meuScript.js", e assim fechamos a tag script.

3º topico: Depuração e console

O console é uma ferramenta importante para depuração, pois permite que você inspecione variáveis, visualize mensagens de erro e execute código JavaScript interativamente. Além disso você pode usar o console como um ambiente interativo para executar código JavaScript diretamente. Isso é útil para testar funções ou experimentar com código em tempo real.

Para abrir o console do navegador, normalmente você pode usar a tecla de atalho F12, Ctrl + Shift + I (ou Cmd + Option + I em sistemas macOS) ou clicar com o botão direito do mouse em qualquer lugar da página e selecionar "Inspecionar" ou "Inspecionar elemento" e, em seguida, navegar para a guia "Console".

O debug de JavaScript no console do navegador é uma habilidade fundamental para os desenvolvedores da web, pois ajuda a encontrar e corrigir erros, melhorar o desempenho e entender o comportamento de um aplicativo web em tempo real.

4º topico: Instruções, Sintaxe e Variáveis

No JavaScript nós seguimos a lógica de que um programa é uma lista de "instruções" a serem "executadas" pela maquina, sendo assim nós chamamos isso de instruções ou em inglês "Statements".

Instruções:

As instruções JavaScript são compostas por: Valores, operadores, expressões, palavras-chave e comentários.

Essas instruções são executadas uma a uma na ordem que foram escritas no código.

Normalmente utilizamos ; para separar ou finalizar instruções, por exemplo a seguir temos:

let x, y, z;
x = 5;
y = 6;
z = x + y;

Nesse exemplos declaramos 3 variáveis, sendo elas x, y, e z, em seguida definimos x como sendo 5, y como sendo 6, e z para a adição de x com y.

Da mesma forma podemos fazer isso em uma única linha:

x = 5; y = 6; z = x + y;

Outro ponto importante é que assim como o Html e o Css, o JavaScript ignora multiplos espaços. Sendo assim podemos utilizar essa funcionalidade para tornar os códigos mais legiveis! Uma boa pratica é separar com espaços operadores (= + - * /). Além disso instruções podem conter quebra de linhas, sendo o melhor canto para isso após um operador.

Também é importante saber que temos agrupamentos de instruções no JavaScript, isso é feito dentro das chaves {}, com o proposito de executar instruções juntas, um lugar que fazemos isso e vamos ver no nosso 5º topico de hoje são as funções!

Para finalizarmos as instruções do JavaScript temos as palavras-chave, essas que são colocadas no inicio de uma instrução do JavaScript, alguns exemplos são: var, let, const, if, switch, for, function, return, try, dentre diversos outros.

Sintaxe:

A sintaxe do JavaScript é definida em dois tipos: Valores fixos, e Valores variaveis. Sendo os Fixos chamados de literais, e os variaveis de variaveis.

As duas regras mais importantes literais são que numeros são escritos com ou sem decimais, no caso de números com decimais esses são separados por ponto e não por virgula. A outra regra é que strings são textos escritos entre aspas simples, ou aspas duplas.

x = 5; y = 6.81; z = 'John Doe';

Outro tópico importante na sintaxe são os operadores matemáticos, que são mais+ (adição), menos- (subtração), asterisco* (multiplicação) e barra/ (divisão).

Em JavaScript o sinal de igual (=) é um operador de "atribuição", não um operador de "igual a". Para esse último utilizamos o sinal de igual duas vezes (==).

let x = 5;
x = x + 5;
x == 10;

Neste exemplo definimos a variável x como sendo 5, em seguida, definimos x como sendo x mais cinco, ou seja, como x era 5 então x mais 5 resulta em x agora possuindo o valor de 10. Em seguida fazemos uma validação de x igual a 10, o que resulta no resultado true (verdadeiro), pois x realmente é igual a 10!

Variáveis:

Na programação, variáveis são usadas para armazenar valores de dados.

No JavaScript usamos as palavras-chave var, let, e const para declarar variáveis, com um simbolo de igual para definir o valor na variável.

Variáveis são declardas de 4 formas diferentes no JavaScript.

A primeira forma é "automática" quando não utilizamos uma palavra-chave, e damos um valor a uma variável, isso acontece devido o JavaScript ser de tipagem dinâmica como vimos anteriormente, mas não é uma boa prática utilizar dessa funcionalidade.

A segunda forma é utilizando a palavra-chave var, essa que era utilizada na antiga versão do JavaScript, e continua existindo para dar suporte a navegadores antigos.

A terceira forma, e a mais convecional, é utilizando as palavras-chave let e const.

Para saber quando se deve usar var, let ou const, pense nas seguintes afirmativas: Sempre declare suas variáveis, sempre use const quando um valor não deve ser alterado, sempre use const quando o tipo não deve ser alterado, apenas use let se você não puder usar const, e apenas use var se você precisa dar suporte a navegadores antigos.

Toda variável possui um identificador, ou nome, que são únicos, podendo ser curto (como x, y e z, vistos anteriormente), ou podendo ser mais descritiveis (como idade, soma, volumeTotal).

Existe uma regra simples para a criação desses identificadores, eles podem conter letras, digitos, travessões e cifrões.

Identificadores não podem começar com números apenas com letras, cifrão e travessão.

Identificadores são sensitiveis, ou seja, uma variável chamada "primeironome" e outra chamada "primeiroNome" (com o N maiúsculo) são variáveis diferentes.

Da mesma forma que palavras-chave também são sensitiveis, ou seja escrever let, com todas letras minusculas é a única forma que o JavaScript vai interpretar tal funcionalidade.

Por fim existem algumas palavras reservadas, como palavras-chave, que não podem ser usadas como identificadores.

5º topico: Definindo Funções em JavaScript

Em JavaScript, você pode criar funções para executar tarefas específicas. Aqui está um exemplo de como definir uma função simples:

function saudacao() {
    console.log("Olá, mundo!");
}

Aqui definimos a função "saudacao()", para que quando chamada ela registre no console de debug a frase "Olá, mundo!"

6º topico:Eventos em JavaScript

Eventos são ações do usuário, como clicar em um botão ou mover o mouse. Você pode responder a esses eventos em JavaScript. Aqui está um exemplo de como ouvir um evento de clique em um botão:

const meuBotao = document.getElementById("meuBotao");
meuBotao.addEventListener("click", function() {
    console.log("Botão clicado!");
});

Aqui temos a criação de uma variável com o identificador meuBotao, que recebe o valor do documento html vinculado pegando especificamente o elemento html com o ID "meuBotao". Em seguida utilizamos a variável criada para fazer um "escutador de evento", esse que ao identificar o evento click vai realizar a function() de registrar no console o texto "Botão clicado!"
