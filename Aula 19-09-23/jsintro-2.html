5º tópico: Definindo Funções em JavaScript

Em JavaScript, você pode criar funções para executar tarefas específicas. Aqui está um exemplo de como definir uma função simples:

function saudacao() {
    console.log("Olá, mundo!");
}

Aqui definimos a função "saudacao()", para que quando chamada ela registre no console de debug a frase "Olá, mundo!"

6º tópico:Eventos em JavaScript

Eventos são ações do usuário, como clicar em um botão ou mover o mouse. Você pode responder a esses eventos em JavaScript. Aqui está um exemplo de como ouvir um evento de clique em um botão:

const meuBotao = document.getElementById("meuBotao");
meuBotao.addEventListener("click", function() {
    console.log("Botão clicado!");
});

Aqui temos a criação de uma variável com o identificador meuBotao, que recebe o valor do documento html vinculado pegando especificamente o elemento html com o ID "meuBotao". Em seguida utilizamos a variável criada para fazer um "escutador de evento", esse que ao identificar o evento click vai realizar a function() de registrar no console o texto "Botão clicado!"

Exercicio desse tópico:

Já no arquivo JavaScript apenas vamos criar a variável que vai armazenar o elemento html com ID meuBotao, e em seguida vamos criar o evento para ser escutado!

7º tópico:Saidas do JavaScript

O JavaScript muitas vezes realiza ações que não são visiveis para os usuários, que apenas recebem o resultado da ação de uma forma mais pratica. O JavaScript possui diferentes formas de "saidas", sendo elas:

• Escrever em um elemento do HTML usando o innerHTML.

• Escrever o HTML usando o document.write().

• Escrever em uma caixa de alerta usando alert().

• Escrever no console do navegador usando o console.log().

document.getElementById("demo").innerHTML = 5 + 6;
document.write(5 + 6);
window.alert(5 + 6);
console.log(5 + 6);

8º tópico:Tipos de dados JavaScript

Bem como já vimos nós podemos criar diferentes tipos de váriaveis no JavaScript, mas quais são esses tipos?

Dentre esses tipos temos que saber que há uma segregação por questões práticas. Abaixo temos os tipos:

String - é usado para definir uma serie de caracteres qualquer (pode conter números, letras, simbolos...).

Number - é usado para definir um número qualquer (em JavaScript não existe um número propriamente inteiro, mesmo ao definir um número omitindo a parte decimal, ele ainda é como se fosse um número com decimais, isso pois em outras linguagens separamos os números em int de inteiros, e float/double de decimais).

Boolean - usado para definir um valor verdadeiro ou falso.

Undefined - quando o valor não é definido o tipo também é indefinido.

Object - são tipos de dados que armazenam diversos valores, por isso separamos esse tipo em 3 "subtipos", sendo eles vistos abaixo.

• Object - armazena propriedades e seus valores, e é escrito dentro de chaves {}.

• Array - armazena diversos items nas posições que foram adicionados, separados por vírgulas, e é escrito dentro de colchetes [].

• Date - armazena o valor de uma data, com ano, mês, dia, hora, minutos, segundos, milisegundos


Além desses tipos existem algumas outras definições no JavaScript mais avançadas, como o Bigint, Sets, Maps, Symbol.
